# 🤖 LLM Document Q&A System (RAG)

This project is a Retrieval-Augmented Generation (RAG) based system that allows users to upload a PDF and ask questions from it.

## 🚀 Features
- Upload PDF documents
- Extract and process text
- Generate embeddings using Sentence Transformers
- Perform similarity search using FAISS
- Answer user queries based on document content

## 🛠️ Tech Stack
- Python
- Streamlit
- Sentence Transformers
- FAISS

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py

## 📌 Use Case
This system helps users quickly find answers from large documents using AI.
